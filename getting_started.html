Getting started¶

Obtaining SCoT¶

The SCoT source repository is available at github: https://github.com/scot-dev/scot

Requirements¶

SCoT depends on the packages numpy (http://www.numpy.org/) and scipy (http://scipy.org/). Furthermore, it supports scikit-learn (http://scikit-learn.org/) and matplotlib (http://matplotlib.org/).

The lowest supported versions of these libraries are numpy 1.8.0, scipy 0.13.3, scikit-learn 0.15.0, and matplotlib 1.4.0. Lower versions may work but are not tested.
